# brand-system-skills — v3 transversal audit (2026-06-22)

A nine-workstream plus deep-seam-trace audit of the v3 plugin (scoper + builder skills that turn any brand into a canon, real assets, prototype, /design-sync kit, token spine, and a mandatory keystone .md an AI can BE). This report consolidates 60 raw findings into one deduplicated, severity-ranked ledger.

## Executive verdict

**QUALIFIED NO — the product is conceptually sound and mechanically clean, but it cannot today complete its own headline promise (a faithful, fully-handed-off ANALYZE build) without out-of-band knowledge, and it ships with broken pointers and stale public metadata.**

The load-bearing reasons:

1. **The handoff is not the single sufficient interface for the default ANALYZE path.** Stage-5 applied-design harvest structurally needs a live-URL pointer to "where the brand lives today," but the contract forbids URLs in the manifest three times and gives no carrier — while the builder is simultaneously told to read those pointers. The build must reach out-of-band or fall back to a capture the scoper's own rubric grades non-build-grade. (WS-A / WS-A-deep BLOCKER)
2. **The installed bundle dangles its own "full specification."** SKILL.md + five references cite `dev/v2-build-spec.md` (the §3 pipeline spec, the only locator for the Stage-8 /design-sync re-pin) ~13 times, but `dev/` is gitignored and never shipped. Every fresh install points at a file the user can never open. (WS-G / WS-H / WS-I BLOCKER)
3. **Orphaned contract blocks.** The handoff emits BUILD-MODE and an exhaustive DIMENSION MAP with hard "STOP on unresolved dimension" semantics (the stated anti-determinism mechanism), but the builder's Stage 0 never parses or acts on either. The v0/DEMO degradation path and the project's central anti-determinism gate have no consumer. (WS-A BLOCKER)
4. **The most owner-visible deliverable is partly fabricated.** The keystone's SPEAK few-shot on/off-brand pairs and THINK trade-off rules have no carrier in the handoff WHY, so Stage 8.5 invents voice exemplars and value-priorities the owner never ratified — exactly the brand-truth-invention the scoper is forbidden to do. (WS-A-deep BLOCKER)
5. **Public surface lies about what ships.** Both manifests omit the mandatory keystone, still market the retired "greenfield/brownfield" framing, and pin version 0.2.0 across a shipped v3 major reframe; `canon.json`'s `$schema` points at a non-existent file. (WS-H)

What holds: the mechanical floor passes everywhere (descriptions ≤1024, fences balanced, 13 stages + 8.5 present, tokens parse and aliases fully resolve, brand-leak grep = 0); the anti-determinism rector and the keystone north-star are genuinely load-bearing; the asset-less-skeleton dead-end is mechanically killed; and the seven historical failure classes each map to a live guardrail.
